# Contributing to OHDSI GIS

Thank you for your interest in contributing to the OHDSI GIS Working Group!

## Quick Start

1. **Join the Community**
   - [Sign up for OHDSI Teams](https://forms.office.com/Pages/ResponsePage.aspx?id=lAAPoyCRq0q6TOVQkCOy1ZyG6Ud_r2tKuS0HcGnqiQZUQ05MOU9BSzEwOThZVjNQVVFGTDNZRENONiQlQCN0PWcu)
   - [Join the GIS Working Group](https://forms.office.com/Pages/ResponsePage.aspx?id=lAAPoyCRq0q6TOVQkCOy1ZyG6Ud_r2tKuS0HcGnqiQZUOVJFUzBFWE1aSVlLN0ozR01MUVQ4T0RGNyQlQCN0PWcu)
   - Attend our [weekly Friday meetings](https://teams.microsoft.com/l/meetup-join/19%3af4d776830f504bf3827fe4309156a3c6%40thread.tacv2/1724707871947?context=%7b%22Tid%22%3a%22a30f0094-9120-4aab-ba4c-e5509023b2d5%22%2c%22Oid%22%3a%22457ee911-94c1-4da2-8287-8d8a5a749931%22%7d) at 10 AM US Eastern

2. **Get Oriented**
   - Read our [Developer Hub](https://ohdsi.github.io/GIS/developer.html)
   - Review the [Architecture](https://ohdsi.github.io/GIS/developer-architecture.html)
   - Check out the [Project Board](https://github.com/orgs/OHDSI/projects/26)

3. **Find Something to Work On**
   - Look for [`good first issue`](https://github.com/OHDSI/GIS/labels/good%20first%20issue) labels
   - Check [`help wanted`](https://github.com/OHDSI/GIS/labels/help%20wanted) issues
   - Review [open use cases](https://github.com/OHDSI/GIS/issues?q=is%3Aissue+is%3Aopen+label%3A%22Use+Case%22)
   - Or [propose your own use case](https://github.com/OHDSI/GIS/issues/new?assignees=&labels=Use+Case&projects=&template=use-case.yaml&title=%5BUse+Case%5D%3A+)

## Types of Contributions

We welcome contributions in many forms:
- **Code**: New features, bug fixes, performance improvements
- **Documentation**: Tutorials, guides, API documentation
- **Data Sources**: New transformation recipes, metadata
- **Research**: Use case proposals, validation studies
- **Community**: Answering questions, reviewing PRs, onboarding newcomers

## Code of Conduct

All contributors are expected to follow the [OHDSI Code of Conduct](https://www.ohdsi.org/code-of-conduct/).
